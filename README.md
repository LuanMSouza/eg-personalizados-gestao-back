## 📋 Sobre

API REST que alimenta o dashboard de gestão da EG Personalizados. Endpoints para pedidos, clientes, produtos e relatórios gerenciais.

## 🚀 Como Rodar

**Pré-requisitos:** Node.js 18+

```bash
git clone https://github.com/LuanMSouza/eg-personalizados-gestao-back
cd eg-personalizados-gestao-back
npm install
cp .env.example .env
npm run dev
```

API disponível em `http://localhost:3001`.
